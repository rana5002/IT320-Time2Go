# IT320-Time2Go

A web application that helps users in Riyadh check congestion levels 
at malls, cafes, and public places before they visit.

## Features
- Search locations by name or category
- Select a branch and view congestion details
- Check predicted congestion for a specific date and time
- Save favorite branches (registered users)
- Notifications for saved places (registered users)
